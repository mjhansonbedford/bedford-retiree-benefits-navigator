# Bedford Retirement Benefits Navigator

A Vercel-ready Next.js application for Bedford HR retirement-benefits guidance.

## Features

- Middlesex Retirement vs. MTRS guidance
- Bedford 10-year GIC eligibility screening
- Automatic 65th-birthday calculation from date of birth
- Medicare review date calculated six months before age 65
- Separate spouse age-65 calculation
- Medicare Part D warning for GIC Medicare plans
- Part B reimbursement reminder
- Spouse and dependent guidance
- Retirement timeline and official resource links

## Deploy to Vercel

1. Upload all files in this folder to the root of your GitHub repository.
2. In Vercel, choose **Add New → Project**.
3. Import `mjhansonbedford/bedford-retiree-benefits-navigator`.
4. Vercel should detect Next.js automatically.
5. Click **Deploy**.

No environment variables are required.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Important

This is a planning and educational tool only. Final benefit eligibility, enrollment, deadlines, and plan choices must be confirmed with Bedford Human Resources and the relevant administering agencies.
